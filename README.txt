ChancenSpiel – Vollversion 1.0
--------------------------------------
Dieses Spiel enthält:
- 400+ Karten
- Alle Bildungsphasen mit Wahrscheinlichkeiten
- Modifikator-Folgeeffekte
- Interaktive Ziehmechanik
- Rückwärtssuche + statistischer Vergleich
- Biografieanzeige mit Balkendiagramm

Nutzung:
1. Öffne index.html im Browser.
2. Keine Internetverbindung notwendig.
3. Voll funktionsfähig auf Desktop und Smartphone.
